UNet Model

The UNet model is a type of convolutional neural network designed for biomedical image segmentation. It consists of an encoder-decoder structure with skip connections that help in retaining spatial information.

Usage Instructions

Using the Random Forest Model

To use the Random Forest model, load it with the following code:

import joblib

# Load the model
model = joblib.load('path_to_model_1.pkl')

# Use the model for prediction
predictions = model.predict(X_test)

Using the UNet Model

To use the UNet model, load it with the following code:

from tensorflow.keras.models import load_model

# Load the model
unet_model = load_model('path_to_model_2.pkl')

# Use the model for predictions
predictions = unet_model.predict(X_test)
